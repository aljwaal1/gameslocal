# خطة تطوير GamesLocal

## الهدف
تطبيق ألعاب محلية احترافي للأندرويد، يدعم اللعب:

- على نفس الجهاز.
- ضد الكمبيوتر / الروبوت.
- عبر Wi-Fi / Hotspot.
- لاحقًا عبر Bluetooth.

## ملفات APK

- `apk/gameslocal-flutter.apk` للأجهزة الحديثة.
- `apk/gameslocal-kitkat.apk` للأجهزة القديمة Android 4.4.

## تصميم المشروع

```text
lib/
  core/        الغرف، تعريف الألعاب، الحالة العامة
  design/      الألوان، البطاقات، النمط البصري
  network/     الاتصال المحلي لاحقًا
  bot/         أساس الروبوت المشترك
  games/       كل لعبة في مجلد مستقل
```

## الألعاب المطلوبة

### 1. إكس أو XO
- حالة حالية: موجودة وقابلة للعب.
- المضاف حاليًا:
  - وضع لاعب ضد لاعب.
  - وضع ضد الكمبيوتر.
  - تمييز خط الفوز.
  - لوحة نتائج للجلسة: X / O / تعادل.
  - زر جولة جديدة وزر تصفير النتائج.
- المطلوب القادم:
  - حفظ النتائج بعد إغلاق التطبيق.
  - إضافة مستويات صعوبة للكمبيوتر.
  - إضافة مؤثر صوتي خفيف عند الحركة والفوز.

### 2. الضامة
- حالة حالية: موجودة كبداية.
- المطلوب القادم:
  - تحسين الرسومات.
  - تقوية الروبوت.
  - إضافة مستويات صعوبة.
  - ربطها باللعب عبر الشبكة.

### 3. الشطرنج
- المطلوب:
  - لوحة احترافية.
  - قطع واضحة.
  - قوانين الحركة.
  - كش / كش مات.
  - روبوت لاحقًا.

### 4. الدومينو
- المطلوب:
  - قطع دومينو احترافية.
  - 2 إلى 4 لاعبين.
  - توزيع عشوائي.
  - دور اللاعب.
  - روبوت لاحقًا.

### 5. الشدة
- المطلوب:
  - أوراق لعب احترافية.
  - 2 إلى 4 لاعبين.
  - نظام توزيع.
  - قواعد قابلة للتعديل.

### 6. السراقة
- المطلوب:
  - تحديد القواعد النهائية.
  - طاولة وأوراق.
  - روبوت لاحقًا.

## خطة العمل المتدرجة

1. تثبيت التصميم العام.
2. إكمال شاشات الألعاب كلها كبداية قابلة للتشغيل.
3. تطوير القوانين لعبة لعبة.
4. إضافة الروبوت لكل لعبة.
5. إضافة Wi-Fi / Hotspot.
6. إضافة Bluetooth.
7. تحسين الجرافيك والأصوات.

## آخر تحديث عملي

- تم تحسين لعبة XO بإضافة لوحة نتائج داخل الجلسة.
- تم توثيق XO كلعبة مستقلة ضمن الخطة.
- الأولوية التالية: حفظ نتائج XO محليًا أو تحسين روبوت الضامة.

## مبدأ مهم
كل لعبة يجب أن تبقى مستقلة داخل مجلدها. ممنوع تعديل لعبة وتخريب لعبة أخرى.
