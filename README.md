# Games Local

تطبيق ألعاب محلية للأندرويد مبني بـ Flutter.

## الفكرة

كل لعبة داخل مجلد مستقل حتى نستطيع تعديل كل لعبة لوحدها بدون التأثير على باقي الألعاب.

## الهيكل

```text
lib/
  core/
    game_definition.dart
    game_room.dart
    network/
      local_network_core.dart
      local_wifi_transport.dart
      network_message.dart
  games/
    checkers/
      checkers_game.dart
    battle/
      battle_mode_screen.dart
    cards/
      cards_game.dart
    checkers/
      checkers_game.dart
    chess/
      chess_game.dart
    chicken/
      chicken_game.dart
    domino/
      domino_game.dart
      domino_four_player_game.dart
    xo/
      xo_game.dart
```

## النسخة الحالية V1

- واجهة عربية RTL.
- قائمة ألعاب: الضامة، الشطرنج، الدومينو، الشدة / السراقة.
- غرفة مشتركة قبل الدخول للعبة.
- لعبة الضامة تعمل محليًا على نفس الجهاز، وتم تجهيز نقل حركتها عبر Wi-Fi / Hotspot.
- إكس أو والضامة تدعمان اللعب المحلي والشبكة المحلية.
- الدومينو والشدة تعملان ضد الكمبيوتر.
- الشطرنج يعمل محلياً للاعبين، ولعبة الدجاجة تتضمن متجر خلفيات.
- النسخة الحديثة تُبنى منفصلة في `apk/modern/gameslocal-flutter.apk`.
- نسخة Android 4.4 Java تُبنى منفصلة في `apk/legacy/gameslocal-kitkat.apk`.

## حالة الإصدار الحالي

- واجهة عربية RTL مع إعدادات محفوظة وصوتيات اختيارية.
- إكس أو، الضامة، Battle Mode، الدومينو، الشدة/السراقة، الشطرنج، ولعبة الدجاجة.
- اللعب المحلي والروبوت، مع دعم LAN للألعاب التي تعرض خيار الشبكة من غرفة اللعبة.
- الدومينو: لاعبان عبر الشبكة أو الروبوت، ووضع منفصل لأربعة لاعبين محليًا على الجهاز نفسه.
- الشطرنج: نقلات قانونية، كش وكش مات وتعادل، ترقية البيدق، تراجع، وتبييت قانوني.
- الضامة: أكل إجباري وسلاسل أكل ونتيجة مباراة؛ ونسخة KitKat تفرض الأكل وتنهي الجولة عند انعدام الحركات.
- الشدة/السراقة: روبوت أو لاعب عبر الشبكة، مع استعادة حالة الجولة للضيف.
- Battle Mode: خرائط وقدرات ومزامنة LAN.
- لعبة الدجاجة: تقدم وخلفيات وإنجازات محفوظة.

## التحقق

- بناء Flutter يشغّل الاختبارات والتحليل ثم ينتج APK حديثًا.
- بناء KitKat Java مستقل ويُشغّل فقط عند تغيير ملفات النسخة القديمة.
- بقي الاختبار اليدوي على جهازين فعليين خطوة إصدار موصى بها للتحقق من الشبكة المحلية؛ اللعب عبر الإنترنت ليس جزءًا من هذا الإصدار.

## فصل نسختي Android

- مشروع Flutter الحديث مستقل ولا يبني ملفات `android_kitkat_java`.
- مشروع KitKat Java مستقل ولا يستورد Flutter أو ملفاته.
- لكل نسخة Workflow واسم APK ومجلد إخراج منفصل.
